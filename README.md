# Friends Favorites Project
## Overview
Created a database using a sample size of my friend group to compare favorite food type and boba drinks, their respective costs and total costs and to see if there were any trends in the data.

## Softwares Used
1. SQL
2. XAMPP
4. Tableau Public
5. Visual Studio Code

# SQL/Xampp
Utilizied SQL and XAMPP to create the database. Organized the tables to replicate a real business scenario eg. a table containing only the boba name, id, and costs and then joining it with another table that has the friend_id to show what each person ordered. Integrated JOIN clauses with subqueries to analyze data wanted within one query eg.
